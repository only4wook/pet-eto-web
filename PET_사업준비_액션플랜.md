# P.E.T 사업 준비 — 대욱님 구체적 액션플랜
> "사람을 연결할 준비"를 만드는 단계별 가이드

---

## 현재 상태 진단

**기술(웹)**: 90% 완료 → 더 이상 코딩할 필요 거의 없음
**사업(매칭)**: 0% → 여기에 집중해야 함

---

## 1단계: 파트너 확보 (이번 주 ~ 2주)

### 방법 A: 지인 네트워크 (가장 빠름)
1. 반려동물 키우는 지인 중 **시간 여유 있는 사람** 찾기
   - 대학원 동기, 동아리, 학교 주변
   - "테스트로 한번만 도와줘" → 부담 없이 시작
2. 반려동물관리사 교육 같이 듣는 수강생에게 제안
3. 수의학과/동물자원학과 학생 → 알바 형태로 제안

### 방법 B: 온라인 모집 (1~2주)
1. **당근마켓** 글 올리기:
```
[고양시] 반려동물 펫시터 파트너 모집 🐾

반려동물 긴급 돌봄 서비스 P.E.T에서
검증된 펫시터 파트너를 모집합니다.

✅ 반려동물 양육 경험 1년 이상
✅ 고양시/일산 거주
✅ 주 2~3회 출동 가능

💰 건당 수익: 2~5만원
📞 문의: 카카오톡 채널 "P.E.T 펫에토"
```

2. **반려동물 커뮤니티** (네이버 카페, 에브리타임)에 홍보

### 방법 C: 기존 펫시터 영입 (2~3주)
1. 도그메이트, 펫플래닛 등에서 활동 중인 개인 펫시터에게 DM
2. "우리 플랫폼에서 추가 수입 가능" 제안
3. 동물병원/펫샵에 명함 놓기

### 파트너 최소 요건:
- [ ] 반려동물 양육 경험 1년+
- [ ] 신분증 확인
- [ ] 대면 면접 (대표님 직접)
- [ ] 시범 케어 1회

## 2단계: 첫 10건 매칭 (2~4주)

### 무료 체험 이벤트
```
🐾 P.E.T 무료 체험 이벤트!

고양시/일산 반려인 선착순 10명
2시간 무료 돌봄 체험

✅ 검증된 펫시터 방문
✅ 실시간 사진 보고
✅ 체험 후 후기 작성 시 5,000원 상품권

신청: https://pet-eto.vercel.app
카톡: P.E.T 펫에토
```

### 홍보 채널 (비용 0원):
1. **당근마켓** — 고양시/일산 동네 홍보 (무료)
2. **인스타그램** — #고양시반려동물 #일산펫시터 해시태그
3. **네이버 카페** — 고양시 맘카페, 반려인 카페
4. **에브리타임** — 한양대 게시판
5. **카카오톡 오픈채팅** — 고양시 반려인 단톡방

### 매칭 프로세스 (수동 — 카톡):
1. 고객이 카카오톡 채널로 문의
2. 대표님이 직접 상담 (날짜, 시간, 반려동물 정보)
3. 적합한 파트너 배정
4. 파트너가 고객 집 방문
5. 실시간 사진/영상 보고 (카톡으로)
6. 케어 완료 후 계좌이체 or 카카오페이
7. 후기 요청

## 3단계: 수익화 (1~2개월)

### 가격 체계 (사이트에 이미 표시됨):
| 서비스 | 가격 | 비고 |
|--------|------|------|
| 기본 돌봄 | 3만원/3시간 | 방문, 급식, 산책 |
| 야간 돌봄 | 4만원/3시간 | 20시 이후 |
| 1일 돌봄 | 8만원/8시간 | 장시간 외출 시 |
| 1박 돌봄 | 12만원 | 출장/여행 |
| 산책 대행 | 1.5만원/1시간 | 산책만 |

### 수수료 구조:
- 파트너에게 **70~80%** 지급
- P.E.T 수수료: **20~30%**
- 기본 3만원 매칭 → P.E.T 수익 6,000~9,000원

### 월 매출 시뮬레이션:
| 매칭 건수/월 | 건당 평균 | 월 매출 | P.E.T 수익 |
|-------------|----------|---------|-----------|
| 10건 | 4만원 | 40만원 | 8~12만원 |
| 30건 | 4만원 | 120만원 | 24~36만원 |
| 50건 | 4만원 | 200만원 | 40~60만원 |
| 100건 | 4만원 | 400만원 | 80~120만원 |

## 5단계: 법적 준비 체크리스트

- [ ] **사업자등록** (간이과세 → 일반과세)
  - 업종: "동물 관련 서비스업" 또는 "온라인 정보 중개업"
  - 세무서 또는 홈택스에서 신청 (무료, 1일)
- [ ] **통신판매업 신고** (온라인 중개)
  - 관할 구청에서 신고 (무료)
- [ ] **배상책임보험 가입**
  - 반려동물 위탁 중 사고 시 배상 (연 20~50만원)
  - KB손보, 삼성화재 등에서 가능
- [ ] **카카오톡 비즈니스 채널 전환**
  - 사업자등록 후 → 채널 인증 → "사칭 주의" 경고 제거
- [ ] **개인정보보호 자체 점검** (사이트에 이미 약관 있음 ✅)
