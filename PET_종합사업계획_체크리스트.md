## 8. 법적 문제 / 책임 / 분쟁 방안

### 법적 구조

| 항목 | 내용 |
|------|------|
| **P.E.T의 역할** | 중개 플랫폼 (직접 서비스 제공자 아님) |
| **파트너의 지위** | 프리랜서 (근로자 아님) → 최저임금법 미적용 |
| **책임 범위** | P.E.T는 매칭 중개만, 케어 중 사고는 파트너-고객 간 계약 |
| **필요 보험** | 배상책임보험 (파트너별 가입 or P.E.T 단체 가입) |

### 분쟁 처리 프로세스

```
1. 고객 불만 접수 (카카오톡/사이트)
   ↓
2. 대표가 양측 의견 청취 (24시간 이내)
   ↓
3. 증거 확인 (사진/영상/채팅 기록)
   ↓
4. 합의 중재 (환불/재서비스/배상)
   ↓
5. 합의 불성립 시 → 소비자분쟁조정위원회/법률 상담
```

### 주요 분쟁 사례 & 대응

| 사례 | 대응 |
|------|------|
| 반려동물 부상 | 즉시 동물병원 이송 + 치료비 선 지급 + 보험 처리 |
| 물건 파손/분실 | 파트너 배상 (보험 처리) |
| 반려동물 탈출 | 즉시 수색 + 동물보호관리시스템 신고 + 보상 |
| 파트너 무단 취소 | 대체 파트너 즉시 배정 + 고객에게 할인 쿠폰 |
| 고객 무단 취소 | 24시간 전: 전액 환불, 당일: 50% 환불 |
