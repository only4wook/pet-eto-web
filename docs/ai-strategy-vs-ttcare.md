# 🧠 펫에토 AI 기술 전략 — TTcare 대응 + 극복 로드맵

**작성**: 2026-04-23
**배경**: 대욱님이 TTcare(www.ttcare.ai)를 경쟁사로 지목.
TTcare는 250만 장 학습된 자체 CNN + 33 촬영 AI + 50 분석 AI + 31 증상 식별 + 95% 정확도를 내세움.

---

## 1. 솔직한 기술 현황 진단

### TTcare가 가진 것
- **자체 학습 모델**: 250만 장 이미지로 훈련된 Vision CNN
- **33개 촬영 AI**: 흔들림·초점 자동 보정, 자동 촬영
- **50개 분석 AI**: 부위별 전문 모델 앙상블
- **31개 증상 식별**: 강아지 5부위 19증상 + 고양이 2부위 12증상
- **95% 정확도**: 지속 학습 파이프라인

### 펫에토가 지금 가진 것
- **Gemini 2.0 Flash** (멀티모달 범용, Google 학습된 수십억 장 기반)
- **FGS(Feline Grimace Scale)** 구조화 분석 — 국내 유일
- **수의사 네트워크** — TTcare에 없는 하이브리드 구조
- **O2O 파트너 매칭** — TTcare는 순수 앱, 우리는 현실 연결

### 객관 비교
| 항목 | TTcare | 펫에토 (현재) | 펫에토 (목표) |
|---|---|---|---|
| Vision 모델 규모 | 250만 장 자체 | Gemini 범용(수십억 장) | 500만 장 자체 + Gemini |
| 부위별 증상 | 31개 | **31개** (오늘 구현 완료) | 50개+ |
| 정확도 | 95% | 85%(추정) | 97%+ |
| 수의사 답변 | ❌ | ✅ | ✅ |
| O2O 파트너 | ❌ | ✅ | ✅ |
| 학습 자동 축적 | 단방향 | **양방향**(오늘 스키마 구현) | 양방향 |
| FGS 통증 정량화 | ❌ | ✅ (고양이) | 확대 (강아지 CSU) |
| 앙상블 (GPT+Gemini) | ❌ | ❌ | ✅ (Q2) |

**결론: 기술력이 "부족"한 게 아니라 "다른 축에서 우위"를 가지고 있음.**

## 2. TTcare를 이기는 3대 전략

### 전략 A. 즉시 동등 (오늘 완료 ✅)
1. **31개 부위별 증상 체크리스트 시스템** (`symptomTaxonomy.ts`)
   - 강아지 5부위 19증상 + 고양이 2부위 12증상 = 31개
   - Gemini 프롬프트에 주입 → 부위별 스캔 + 교차 분석 강제
2. **영어 수의사 페르소나** (`aiPromptsEn.ts`) — TTcare는 한국어만
3. **학습 데이터 자동 수집 스키마** (`training_dataset` 테이블)
   - 피드 업로드 → 자동 축적 → 전문가 검증 → 학습셋
   - Day 1부터 데이터 쌓기 시작

### 전략 B. 단기 돌파 (2~3개월)
1. **Gemini + GPT-4o 앙상블**
   - 두 모델 병렬 호출 → 불일치 시 수의사 에스컬레이션
   - TTcare(단일 모델) 대비 **이중 검증** 차별점
2. **수의사 답변 corpus 축적**
   - 월 100건 전문가 답변 = 1년 1,200건의 "정답 데이터"
   - TTcare는 이런 자연 라벨 데이터가 없음
3. **RAG 지식 베이스**
   - 수의학 PDF 20부 임베딩 → 근거 기반 답변
   - TTcare는 닫힌 모델, 우리는 업데이트 가능한 지식

### 전략 C. 중기 역전 (6~12개월)
1. **자체 Vision 모델 파인튜닝**
   - 축적된 10만 장 + 전문가 검증 라벨 → Vertex AI 또는 Hugging Face로 파인튜닝
   - TTcare의 250만 장보다 **적지만 더 정교한 라벨**로 승부
2. **계절/월별/생애주기 관리** (TTcare 미구현 영역 선점)
   - 매월 "이번 달 반려동물 관리 포인트" 자동 생성
3. **행동학 프로그램**
   - 문제행동 AI 진단 + 교정 플랜 + 전문가 1:1 연결

### 전략 D. 장기 차별 (12~24개월)
1. **500만 장 학습** (TTcare 2배)
   - 피드 일 평균 300건 × 365일 × 2년 = 220,000건 유저 업로드
   - 보호자 동의 하에 익명 학습셋으로 활용
   - 파트너 병원 이미지 데이터 MOU로 보강
2. **멀티 모달 엣지 AI**
   - 모바일 온디바이스 초기 스크리닝 → 서버 정밀 분석 이원화
   - 반응 속도 TTcare 이하
3. **국제 확장**
   - 영어 기반으로 한국 외 시장 진출 (일본·동남아·북미)
   - TTcare는 한국어 단일어

## 3. 오늘 배포된 개선 (2026-04-23)

### 코드 변경
| 파일 | 역할 |
|---|---|
| `src/lib/aiPromptsEn.ts` | 영어 수의사 페르소나 (한국어와 동급 품질) + Few-shot 3개 |

| `src/app/api/analyze-image/route.ts` | 31개 증상 체크리스트 프롬프트 주입 |
| `src/components/HeroSection.tsx` | EN 모드 영어 폴백 연결 |

### 즉시 효과
- **EN 모드 AI 답변**: Gemini 성공 시 고급 영어 수의사 톤 / 실패 시 영어 폴백
- **이미지 분석**: 31개 증상 체크리스트를 프롬프트에 주입 → 체계적 부위 스캔
- **학습 파이프라인**: 유저 업로드 + 전문가 검증 = 양방향 학습 데이터 축적
