## 🎯 펫에토 플랫폼 비전 (반드시 기억)

### 1. 존재 이유
"파편화된 펫 정보를 정확히 통합" + "질병 → 진료 → 약 → 비용 사전 가이드" + "실제 수의사·동물학도 전문 피드백"

### 2. 핵심 서비스 5가지
| 서비스 | 핵심 매커니즘 |
|---------|--------------|
| **피드 중심 AI 평가** | 사진/영상 업로드 → AI 분석 → 건강하면 ✅ 이상 / 문제 시 전문가 연계 |
| **전문가 피드백 네트워크** | 수의사·동물학 전공·동물병원 계정 인증 → 직접 답변 (정보 주체 명시: "○○병원 ○○수의사") |
| **펫 이동 서비스** | 제휴 업체 모집 + 신청 란 |
| **호텔링/돌봄 서비스** | 액션캠·실시간 보고·검증된 시터 |
| **3단계 검증** | 신원 → 대면 면접 → 시범 케어 (구체 플로우 문서화 필요) |

### 3. 메인 = 피드 (목표)
현재 히어로+AI 챗이 메인이지만, **최종 메인은 피드가 되어야 함**.
히어로 → AI 챗 → **피드 스트림** → 신뢰 그리드 → How It Works 순서로 재구성 예정.

### 4. AI 수준 목표
"Gemini 2.0 Flash + ChatGPT 최고 수준" → 필요 시 다중 모델 앙상블 (Gemini 메인 + GPT-5 보조)
응답에 반드시 **정보 주체**(수의사명·병원명·출처 URL) 명시하는 구조로 발전.

### 5. 영업 타겟
수의사 / 동물병원 / 수의학교(서울대·건국대·충남대·경상대·전남대·전북대·제주대·강원대·충북대·경북대) / 펫샵
