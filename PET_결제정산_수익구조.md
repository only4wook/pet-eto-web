# P.E.T 결제·정산·수익 구조 설계

---

## 1. 거래 흐름 (돈의 흐름)

```
고객 결제 (10만원)
    ↓
P.E.T 에스크로 예치 (토스페이먼츠/카카오페이)
    ↓ 케어 완료 확인
정산 분배:
    ├─ 파트너 수익: 75,000원 (75%)
    ├─ P.E.T 수수료: 21,500원 (21.5%)
    └─ PG 수수료: 3,500원 (3.5%)
```

---

## 2. 수수료 구조

### 2-1. 고객 → P.E.T (결제)
| 서비스 | 고객 결제액 | 포인트 할인 | 실결제 |
|--------|-----------|-----------|--------|
| 기본 3시간 | 30,000원 | 최대 3,000P (10%) | 27,000원~ |
| 야간 3시간 | 40,000원 | 최대 4,000P | 36,000원~ |
| 1일 8시간 | 80,000원 | 최대 8,000P | 72,000원~ |
| 1박 케어 | 120,000원 | 최대 12,000P | 108,000원~ |

**포인트 할인 규칙:**
- 결제액의 최대 10%까지 포인트 사용 가능
- 1P = 1원
- 포인트 적립: 결제액의 3% (다음 이용 시 사용)

### 2-2. P.E.T → 파트너 (정산)
| 항목 | 비율 | 금액 (10만원 기준) |
|------|------|-------------------|
| **파트너 수익** | **75%** | 75,000원 |
| **P.E.T 수수료** | **21.5%** | 21,500원 |
| PG 결제 수수료 | 3.5% | 3,500원 |
| **합계** | 100% | 100,000원 |

### 왜 75%인가?
- 도그메이트, 펫플래닛 등 경쟁사: 파트너에게 70~80% 지급
- 75%면 파트너에게 경쟁력 있는 수익 + P.E.T도 21.5% 확보
- 초기에는 80% 지급으로 파트너 확보 → 안정화 후 75%로 조정 가능

## 4. 파트너 세금 처리 (3.3% 원천징수)

### 4-1. 법적 구조
- 파트너는 **프리랜서(사업소득자)** → 3.3% 원천징수 의무
- P.E.T가 정산 시 3.3% 원천징수 후 지급
- 원천세 = 소득세 3% + 지방소득세 0.3%

### 4-2. 파트너 실수령액 계산
```
고객 결제: 100,000원
├─ PG 수수료: -3,500원
├─ P.E.T 수수료: -21,500원
├─ 파트너 지급 대상: 75,000원
│   ├─ 원천징수 (3.3%): -2,475원
│   └─ 파트너 실수령: 72,525원
```

### 4-3. P.E.T가 해야 할 것
| 의무 | 시기 | 방법 |
|------|------|------|
| 원천징수 3.3% 공제 후 지급 | 매 정산 시 | 통장 이체 |
| 원천징수 세금 납부 | 매월 10일까지 | 홈택스 전자신고 |
| 지급명세서 제출 | 매년 3월 | 홈택스 → 사업소득 지급명세서 |
| 파트너에게 원천징수 영수증 발급 | 요청 시 | 홈택스에서 출력 |

### 4-4. 파트너 세금 안내 (파트너에게 보낼 내용)
```
P.E.T 파트너 세금 안내

1. 파트너 수익은 사업소득(프리랜서)으로 분류됩니다.
2. P.E.T에서 정산 시 3.3% 원천징수 후 지급합니다.
3. 연간 수입이 2,400만원 이하면 단순경비율로 종합소득세 신고합니다.
4. 5월 종합소득세 신고 시 원천징수된 세금을 정산받을 수 있습니다.

예) 월 100만원 수입 시
- 원천징수: 33,000원/월
- 실수령: 967,000원/월
- 5월 종합소득세 신고 시 환급 가능
```

## 5. 에스크로(예치) 시스템

### 5-1. 에스크로란?
- 고객이 결제하면 P.E.T가 보관 (파트너에게 즉시 지급 X)
- 케어 완료 후 고객이 "완료 확인" → 파트너에게 정산
- 분쟁 발생 시 P.E.T가 중재

### 5-2. 구현 방법 (단계별)

**Phase 1: 수동 에스크로 (지금 ~ 3개월)**
```
1. 고객이 카카오페이/계좌이체로 P.E.T 계좌에 입금
2. 케어 완료 후 고객이 카톡으로 "완료 확인"
3. P.E.T가 수수료 제외 후 파트너 계좌에 이체
```
- 비용: 0원
- 단점: 대표님이 직접 해야 함

**Phase 2: 간편결제 연동 (3~6개월)**
- 토스페이먼츠 또는 카카오페이 에스크로 API
- 고객 결제 → 에스크로 예치 → 완료 확인 → 자동 정산
- 비용: PG 수수료 3.5%
- 필요: 사업자등록, 통신판매업 신고

**Phase 3: 자동 정산 (6개월~)**
- 주간/월간 자동 정산 배치
- 파트너 대시보드에서 수익/정산 내역 확인
- 세금 원천징수 자동 계산

### 5-3. 사업자 계좌 준비
- **사업자 통장 개설** (사업자등록 후)
- 고객 입금용 / 파트너 정산용 분리 권장
- 카카오뱅크 사업자 통장: 무료, 빠른 개설

## 6. 포인트 시스템과 결제 연동

### 6-1. 포인트 적립
| 행동 | 적립 | 비고 |
|------|------|------|
| 서비스 이용 | 결제액의 3% | 1만원 이용 → 300P |
| 후기 작성 | +500P | 텍스트 후기 |
| 사진 후기 | +1,000P | 사진 포함 후기 |
| 친구 초대 | +3,000P | 추천인/피추천인 모두 |
| 회원가입 | +100P | 기존 유지 |
| 게시글 작성 | +10P | 기존 유지 |

### 6-2. 포인트 사용
| 사용처 | 한도 | 비고 |
|--------|------|------|
| 서비스 결제 시 할인 | 결제액의 10%까지 | 1P = 1원 |
| 프리미엄 기능 해제 | 추후 | 긴급 매칭 우선권 등 |

### 6-3. 포인트 정책
- 최소 사용: 1,000P 이상부터
- 유효기간: 적립일로부터 1년
- 현금 전환: 불가 (서비스 할인만)

## 7. P.E.T가 돈을 버는 5가지 방법

| # | 수익원 | 예상 비율 | 시작 시기 |
|---|--------|----------|----------|
| 1 | **중개 수수료 (21.5%)** | 70% | 즉시 |
| 2 | **제휴 병원/미용실 광고** | 10% | 3개월 후 |
| 3 | **프리미엄 구독** (월정액) | 10% | 6개월 후 |
| 4 | **제휴 커머스** (사료/용품 판매) | 5% | 9개월 후 |
| 5 | **데이터 기반 보험 연계** | 5% | 12개월 후 |

### 상세:
1. **중개 수수료**: 모든 매칭의 21.5% (핵심 수익)
2. **제휴 광고**: AI 병원 추천에 제휴 병원 우선 노출 → 월 5~20만원/병원
3. **프리미엄 구독**: 월 15만원 → 주 2회 산책 + 긴급 우선 매칭
4. **커머스**: 파트너가 추천하는 사료/용품 → 판매 수수료
5. **보험**: 매칭 데이터 기반 맞춤형 펫보험 연계 → 가입 건당 커미션
