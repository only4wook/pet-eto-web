## 1. 경쟁사 심층 분석 + 차별점

### 직접 경쟁사 (펫시터 매칭)

| 서비스 | 설립 | 규모 | 핵심 | P.E.T 대비 약점 |
|--------|------|------|------|----------------|
| **도그메이트** | 2015 | 15만건+ 매칭 | 방문돌봄 예약 | 2025년 째깍악어에 인수, 독립성 상실. 긴급매칭 없음 |
| **와요(WAYO)** | - | 10만+ DL | 산책/방문돌봄 | GPS 있지만 긴급 특화 없음. 예약 중심 |
| **펫플래닛** | - | 20만 회원 | 가정위탁 | 규제샌드박스 승인. 방문돌봄 약함 |
| **펫봄** | - | 소규모 | 근거리 매칭 | 브랜드 인지도 낮음. AI 기능 없음 |

### 간접 경쟁사 (커머스/헬스)

| 서비스 | 2024 매출 | 핵심 | P.E.T와의 관계 |
|--------|-----------|------|---------------|
| **핏펫(FitPet)** | 1,631억원 | AI 건강체크, 커머스 | 돌봄 매칭 미진출. 향후 제휴 가능 |
| **펫프렌즈** | 1,500억 거래액 | 새벽배송 | 커머스 전문. 서비스 플랫폼 아님 |

### P.E.T 압도적 차별점 (5가지)

1. **긴급 특화**: "10분 매칭"은 업계 유일. 경쟁사는 모두 24시간+ 예약 방식
2. **AI 건강 분석**: 22개 카테고리, 300+ 키워드 — 핏펫 외에 이 수준의 AI 없음
3. **바디캠(액션캠) 의무 착용**: "전 과정 녹화 = 사각지대 제로" — 업계 최초
4. **콘텐츠 플라이휠**: 위키 28종 + 커뮤니티 + AI → 유입 → 전환 → 매칭
5. **무재고 Asset-light**: 0원 시작, 재고/물류 비용 없음

## 2. 특허권 조사 + 가출원 전략

### KIPRIS 조사 결과
- "반려동물 긴급 돌봄 매칭" 관련 등록 특허: **미확인 (블루오션)**
- "펫시터 매칭 플랫폼" 관련: 일반 매칭 기술 특허만 존재
- GPS + 바디캠 + 긴급매칭 결합 특허: **미존재**

### 특허 출원 가능한 BM

**발명의 명칭 (안):**
"GPS 위치 데이터와 바디캠 영상을 연동한 1인 가구 대상 긴급 반려동물 돌봄 매칭 시스템 및 방법"

**청구항 핵심:**
1. 사용자 GPS와 펫시터 GPS를 실시간 비교하여 최근접 매칭
2. 바디캠 실시간 스트리밍 → 보호자 앱 전송
3. AI 건강 분석 결과 기반 펫시터 역량 매칭 (증상별 전문 펫시터 배정)
4. 긴급도 스코어링 알고리즘 (증상 심각도 x 시간 x 거리)

### 가출원 절차 & 비용

| 항목 | 비용 | 비고 |
|------|------|------|
| 관납료 (전자출원) | 46,000원 | 특허청 |
| 변리사 수수료 | 50~150만원 | 명세서 수준에 따라 |
| **가출원 총비용** | **약 50~200만원** | |
| 정식출원 전환 | +150~400만원 | 12개월 내 필수 |

**권장**: MVP 알고리즘 설계문서(PPT)만으로 가출원 가능. **즉시 진행 권장**.

## 4. 고객 관점 필수 혜택 패키지

### 핵심 원칙: "할인보다 안심"

| 등급 | 혜택 | 목적 |
|------|------|------|
| **기본** | 돌봄 시작/종료 카톡 알림 + 사진 3장 | 기본 안심 |
| **프리미엄** | 바디캠 실시간 스트리밍 + GPS 산책 경로 | 완전 투명성 |
| **VIP** | 돌봄 후 1분 하이라이트 영상 + 건강 리포트 | 감성 + 데이터 |

### 고객이 진짜 원하는 것 (인터뷰 기반 예상)

1. **실시간 카톡 사진/영상 전송** (95% 필수)
2. **펫시터 신원 검증 투명 공개** (90%)
3. **배상책임보험 가입 증명** (85%)
4. **돌봄 후 체크리스트 (밥, 물, 화장실, 건강상태)** (80%)
5. **24시간 긴급 연락 채널** (75%)
6. **첫 이용 할인 쿠폰보다 "무료 시범 돌봄 15분"이 효과적** (신뢰 확보)

## 5. 펫시터 자격/역량 + 액션캠

### 필수 자격

| 항목 | 필수/우대 | 비고 |
|------|-----------|------|
| 반려동물 양육 경험 1년+ | 필수 | 면접 시 확인 |
| 범죄경력조회서 제출 | 필수 | 성범죄, 아동학대 이력 조회 |
| 반려동물관리사 자격증 | 우대 | 한국반려동물관리사협회 |
| 동물보건사 면허 | 우대 | 수의사법 기반 |
| 반려동물행동교정사 | 우대 | 문제행동 대응 |
| 응급처치 교육 이수 | 필수 | P.E.T 자체 교육 (2시간) |

### 3단계 검증 프로세스

1. **서류심사**: 이력서 + 자격증 + 범죄경력조회서
2. **대면 면접**: 대표 직접 면접 (30분) — 동물 취급 시뮬레이션
3. **시범 케어**: 대표 반려동물(고양이 2마리)로 실습 (1시간)

| 항목 | 사양 | 비용 |

| 착용 방식 | 가슴 스트랩 클립형 | 1만원/개 |
| 녹화 | 1080p, 연속 2시간+ | microSD 포함 |

## 8. 법적 문제/책임/분쟁 방안

### 핵심 법률

| 법률 | 적용 사항 |
|------|-----------|
| 동물보호법 | 동물학대 금지, 위탁관리업 등록 |
| 전자상거래법 | 온라인 중개 플랫폼 의무 |
| 개인정보보호법 | 고객/반려동물 정보, 바디캠 영상 관리 |
| 민법 | 손해배상 책임 |

### 분쟁 시나리오 & 대응

| 시나리오 | 책임 소재 | 대응 방안 |
|----------|-----------|-----------|
| 돌봄 중 반려동물 부상 | 펫시터 과실 → 보험 처리 | 배상책임보험 필수 가입 |
| 반려동물 기저질환으로 사망 | 보호자 미고지 → 면책 | 이용약관에 기저질환 고지 의무 명시 |
| 펫시터가 물품 파손 | 펫시터 책임 | 손해배상 조항 + 보험 |
| 반려동물이 펫시터 부상 | 보호자 책임 (민법 759조) | 물림 이력 고지 의무 |
| 바디캠 개인정보 유출 | 플랫폼 책임 | 24시간 후 자동 삭제, 암호화 저장 |

### 필수 법적 문서 (사업자등록 후)

1. **이용약관** (전자상거래법 필수)
2. **개인정보처리방침** (개인정보보호법 필수)
3. **펫시터 파트너 계약서** (업무위탁 계약)
4. **돌봄 동의서** (고객 서명 — 건강 정보, 면책 조항)
