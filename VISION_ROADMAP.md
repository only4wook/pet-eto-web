# 펫에토 비전 반영 현황 + 로드맵
> 업데이트: 2026-04-19
> 대욱님이 공유한 핵심 비전을 15개 항목으로 쪼개고 현재 상태를 체크

---

## 🎯 비전 요약 (대욱님 원문 기반)

1. **정확·통합된 펫 정보** — 파편화된 정보 → 한 곳에서 검증된 답변
2. **증상 → 진료 → 약 → 비용 사전 가이드** — 병원 가기 전 미리 시뮬레이션
3. **실제 수의사/동물학 전공자 피드백** — 전문가 인증 커뮤니티
4. **메인 = 피드 (사진/영상 + AI 평가)** — 건강하면 ✅ / 문제 있으면 전문가 연계
5. **체크박스로 전문가 답변 요청** — 등록 업체가 피드백 제공
6. **정보 주체 명시** — "○○ 동물병원 ○○ 수의사" 답변 형식
7. **펫 이동 서비스** — 제휴 업체 + 신청 란
8. **돌봄·호텔링 서비스** — 액션캠·실시간 보고·출장 대체
9. **3단계 검증 구체화** — 시터·업체 검증 플로우 문서화
10. **신뢰 가는 시스템** — 결제·환불·사고 대응
11. **사용자가 계속 올 이유** — 습관 형성
12. **AI 수준 = Gemini + GPT 최고 수준** — 다중 모델 앙상블
13. **수의사·수의학교·동물병원·펫샵 영업 홍보**
14. **모집·등록·검증 시스템** — 파트너 온보딩
15. **텔레그램 자동 안내** — 작업 완료 알림 자동화

## 🗺 권장 로드맵 (우선순위순)

### Phase 1 — 메인 = 피드 전환 (1~2일)
**목표**: 비전의 "메인 = 피드" 실현 + 피드가 가장 잘 쓰이게

- [ ] 홈 페이지에 **피드 스트림** 삽입 (히어로 다음, AI 챗 옆)
- [ ] 피드 업로드 플로우 개선: 사진 선택 → AI 분석 → **전문가 답변 요청 체크박스** → 피드 발행
- [ ] 피드 카드에 AI 분석 요약 + 전문가 답변 공간 카드 추가
- [ ] 피드 무한 스크롤 + 필터(건강/주의/긴급)

### Phase 2 — 전문가 인증 시스템 (2~3일)
**목표**: 수의사/동물학 전공자가 진짜 답변할 수 있는 구조

- [ ] **전문가 계정 신청** 페이지 `/expert/apply`
  - 수의사 면허번호 / 학교 / 병원명 / 경력 / 면허증 이미지 업로드
- [ ] 관리자 승인 페이지 (기존 `/admin`에 섹션 추가)
- [ ] 승인된 유저에게 `role: "vet" | "student" | "clinic"` 자동 부여
- [ ] 전문가가 단 답변에는 **"○○ 동물병원 ○○ 수의사" 뱃지** + 면허 검증 표시
- [ ] 피드/커뮤 댓글 공간에 전문가 답변 최상단 우선 노출

### Phase 3 — 파트너 서비스 3종 분리 (2~3일)
**목표**: 이동·호텔링·돌봄을 각자의 신청·매칭·검증 플로우로

- [ ] `/partner/transport` — 펫 이동 서비스 (업체 등록 + 보호자 신청)
- [ ] `/partner/hotel` — 호텔링 (업체 등록 + 예약 시스템)
- [ ] `/partner/sitter` — 방문 돌봄 (액션캠 동의 + 실시간 보고 약속)
- [ ] **3단계 검증 상세 가이드 페이지** `/partner/verification`
  - Step 1: 신원 확인 (주민등록 사본·은행계좌)
  - Step 2: 대면 면접 (화상 15분)
  - Step 3: 시범 케어 (1시간·감독 하에)

### Phase 4 — AI 품질 최고 수준 (3~5일)
**목표**: Gemini + GPT 앙상블 + RAG + 정보 주체 명시

- [ ] OpenAI GPT-5 API 추가 연동 (`/api/ai-chat-gpt`)
- [ ] Ensemble 라우팅: 첫 응답 Gemini → 의료 키워드면 GPT 교차 검증 → 차이 있으면 두 답변 비교 노출
- [ ] 수의학 논문·공식 가이드라인 RAG DB 구축 (일단 수작업으로 20~50개 문서 시드)
- [ ] AI 응답에 "참고 출처: [링크]" / "검토 수의사: ○○○" 자동 삽입 (가능할 때)
- [ ] 응급 증상 감지 시 **24시 응급 동물병원 GPS 자동 추천**

### Phase 5 — 영업·온보딩 툴 (1~2일)
**목표**: 수의학교·동물병원·펫샵 아웃리치 효율화

- [ ] `/for-vets` 전문가 전용 랜딩 페이지 (가입 혜택·수수료 정책)
- [ ] 제휴 병원 로고 캐러셀 (피드 하단)
- [ ] 수의학교 10개 대학 타겟팅 페이지
- [ ] 제안서 PDF 자동 생성 기능 (추후)

### Phase 6 — 사용자 습관 설계 (ongoing)
- [ ] 피드 알림 시스템 (내 글에 전문가 답변 달리면 PUSH)
- [ ] 주간 건강 리포트 자동 발송
- [ ] 반려동물 생일·예방접종 리마인더
