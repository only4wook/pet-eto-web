## 1. AI 답변 퀄리티 C-level 대전환 (메인)

### 전략
기존 룰기반(if-else 수십 개) → **Gemini 2.0 Flash 기반 대화형 엔진** 전환.
룰기반은 네트워크/쿼터 문제 시에만 작동하는 "폴백"으로 격하.

### 아키텍처
```
User 질문
  ↓
/api/ai-chat (Gemini 2.0 Flash)
  ├─ 성공 → C-level 답변 반환
  └─ 실패 → 룰기반 폴백 (aiKnowledge + 전용 핸들러)
```

### 신규 파일
- `src/lib/aiPrompts.ts` — 공통 페르소나·Few-shot·파라미터 모듈
  - 이 한 파일만 수정하면 사이트 전체 AI 톤 통일
  - `/api/ai-chat`, `/api/analyze-image` 모두 여기서 페르소나 주입

### 프롬프트 스펙 (C-level 기준)

**페르소나**: 서울대 수의대 박사, 20년 임상, 행동학·내과·응급의학 전문

**사고 절차 (내부 CoT)**:
1. 질문에서 종·나이·맥락 추출
2. 긴급도 판단 (긴급/주의/관찰/정상/정보)
3. 표면 질문 vs 숨은 걱정 구분
4. 3가지 가능성 저울질 후 최고 확률 답
5. 질문 복잡도에 맞는 길이·포맷

**답변 구조 (필수)**:
1. 공감 한 줄 ("놀라셨겠어요", "걱정되시죠")
2. 심각도 명시 (🚨긴급 / ⚠️주의 / 💡관찰 / ✅정상 / ℹ️정보)
3. 원인 3~5가지 (모호한 표현 금지)
4. 즉시 대처 (번호 목록)
5. 병원 기준 (언제 가야 하는지)
6. 한국 비용 시세
7. 후속 질문 유도

**절대 금지**:
- "수의사와 상담하세요"만으로 종결
- "~일 수 있어요" 모호 표현
- 영어 축약어 남발
- 공포 조장
- "죄송합니다" 방어적 표현

**Few-shot 예시 5개 삽입** (프롬프트에 직접):
- 고양이 인덕션 화상 → 1~3도 판단·찬물 10분·병원 기준·15~30만원
- 고양이 염색 가능 → 의학 3가지·동물보호법·안전 대안
- 고양이 + 햄스터 합사 → 반응속도 0.1초·이중 케이지·대안
- 고양이 초콜릿 섭취 → 테오브로민 기준·시간표·치료비 5~150만원
- 고양이 강아지 싸움 → 심각도·원인 5개·72시간 분리·4주 재합사

**파라미터**:
- temperature: 0.6 (일관성)
- topP: 0.92, topK: 40
- maxOutputTokens: 2500 (풀 답변 여유)
- 대화 히스토리: 최근 8턴

## 2. 모바일 카테고리 레이아웃 재구성

### 문제
`/community` 테이블(7열 고정 너비)이 375px 모바일에서 깨짐

### 해결 (globals.css)
- `table` → `display: block` + 카드형 리스트로 변환
- `thead` 숨김
- 각 `tr` → 둥근 카드 (12px 패딩, 1px 테두리, 10px radius)
- 열 재배치:
  1. `[말머리]` 배지 상단 인라인
  2. **제목** 블록, 전체 폭, ellipsis 없음 (가독성)
  3. 닉네임·날짜 하단 인라인

## 3. 닉네임 개인정보 보호 강화

### 문제
피드/커뮤니티에서 일부 유저가 `gsh941025` 같은 이메일 로컬파트 형태 닉네임으로 노출 → 생년월일·아이디 추정 가능

### 해결
**`safeNickname()` 강화** (src/lib/utils.ts):
1. 이메일(@) 차단
2. UUID 정규식 차단
3. 순수 숫자 6자+ 차단
4. 16자+ 헥사 해시 차단
5. kakao_/google_/naver_ 접두사 차단
6. 24자 초과 차단
7. **영문+숫자4자리+ 패턴 차단** (gsh941025·kim0825류) — 한글 포함 닉네임은 예외

**`AuthProvider.tsx` 자동 보정**:
- 로그인 시 `nickname === email.split("@")[0]` 이면 익명 닉네임으로 DB 일회성 교체
- 예: gsh941025 → 귀여운펫대디6617 (해시 결정론적)
