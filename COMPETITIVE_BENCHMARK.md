## 2. 각 경쟁사의 "좋은 점" — 벤치마크 리스트

### 2.1 TTcare에서 배울 것
- ✅ **카드뉴스 형식 피드** (틱톡처럼 하나씩) — 우리도 이번에 `FeedSwipeView`로 구현 ✅ 완료
- ✅ **반려동물 프로필에 건강 카테고리 체크** (피부·안구·치아·비만·관절 등)
- ✅ **분석 결과에 "연령대 평균 대비" 시각화** (너의 고양이는 동년배 평균보다 00%)
- ⭐ **홈 화면 동화책 같은 톤** (아이콘 친근, 폰트 둥글)

### 2.2 Sylvester.ai에서 배울 것
- ✅ **FGS(Feline Grimace Scale)** 기반 통증 수치화 — 고양이 얼굴 5개 지표로 점수
- ✅ **연구 논문 기반 설명** — "이 분석은 ○○ 대학 수의학과 논문 기반" 출처 명시
- ⭐ **응급도 자동 판단** — 7점 이상이면 병원 방문

### 2.3 Feline Grimace Scale에서 배울 것
- ✅ **객관적 점수화 모델** — 주관적 "아파 보여요" 대신 "귀 위치 0~2점, 눈 감음 0~2점"
- ⭐ **수의학 표준 척도 도입**으로 전문성 증명

### 2.4 Petriage에서 배울 것
- ✅ **증상 체크리스트** → **응급도 신호등** (빨강·노랑·초록) → **수의사 추천**
- ✅ **내 동물병원 지정 기능** (자주 가는 병원 1곳 등록 → 응급 시 바로 연결)
- ⭐ **추적 관리** — 증상 체크를 매일 기록 → 변화 그래프

### 2.5 윤샘AI에서 배울 것
- ✅ **수의사 개인 브랜드 아바타** — "실제 사람"이 답변한다는 신뢰
- ✅ **유튜브·인스타 연동 컨텐츠** — 콘텐츠 → 서비스 자연 유입
- ⭐ **콘텐츠 허브** (질병별 아티클 + 영상 + Q&A)

## 3. 펫에토가 이길 수 있는 4가지 차별점

### 🥇 차별점 1: "AI + 전문가 하이브리드" (We win)
**경쟁사는 AI 단독 or 전문가 단독이지만, 펫에토는 AI 1차 분석 → 전문가 검증 2단계 구조**
- TTcare·Sylvester: AI만 (책임 회피형)
- 윤샘AI: 전문가 1인 (규모 한계)
- **펫에토: AI 분석 → 전문가 답변 요청 체크박스 → 수의사가 "○○병원 ○○수의사" 신원 공개로 답변**
- 결과: AI의 속도 + 전문가의 신뢰 모두 확보

### 🥈 차별점 2: "플랫폼 = 생태계" (We win)
**단순 분석 앱이 아닌, 펫시터·이동·호텔링·쇼핑까지 하나로**
- TTcare·Sylvester·Petriage: 분석만
- 펫에토: 분석 → 내원 → 돌봄 → 이동까지 end-to-end
- 결과: 보호자가 여러 앱 쓸 필요 없음 (락인 강력)

### 🥉 차별점 3: "한국 수의사법 컴플라이언스" (We win in KR)
**AI를 "진료 전 참고 가이드"로 선제적 포지셔닝 + 1억 배상책임보험**
- TTcare: 법적 방어 약함
- 해외 서비스: 한국 법규 적용 불가
- 펫에토: 이용약관·UI·AI 프롬프트 전반에 법적 고지 명시
- 결과: 수의사법 위반 논란 원천 차단

### 💎 차별점 4: "가격·비용 예측" (UNIQUE — We're first)
**"내 강아지가 이 증상이면 병원가서 얼마 나와요" 사전 예측**
- 모든 경쟁사: 증상만, 비용 미제공
- 펫에토: 증상 → 의심 질환 → **한국 동물병원 평균 진료비 범위** 자동 제공 (이미 AI 프롬프트에 반영)
- 결과: 보호자가 "가봤다가 비싸면 어쩌지" 불안 해소

## 4. 즉시 도입 추천 (Phase 4~5)

### 도입 1: FGS 기반 통증 수치화 모듈
`/api/analyze-image` 고양이 얼굴일 때:
- 귀 위치 (0 곧게 / 1 애매 / 2 뒤로 누움)
- 눈 감음 (0 보통 / 1 반쯤 / 2 꽉 감음)
- 입 모양, 수염, 머리 위치 각 0~2
- 합계 7점 이상 = 통증 고위험 → 긴급 병원

**구현 방법**: Gemini Vision 프롬프트에 FGS 규칙 추가하면 바로 가능. 별도 모델 학습 불필요.

### 도입 2: 건강 체크리스트 프로필
- 마이페이지 → 반려동물 → "관리 중인 건강 카테고리" 체크
- 피부·안구·치아·비만·관절·심장·신장 7대 카테고리
- 체크한 카테고리는 피드 업로드 시 AI가 "해당 부위 중심 분석" 강화

### 도입 3: 단골 동물병원 등록
- 사용자 프로필에 "자주 가는 병원 1곳" 등록
- 응급 시 `/ai-chat` 또는 피드에서 AI가 "지금 바로 ○○병원에 연락" 원클릭 링크

### 도입 4: 증상 추적 그래프
- 같은 반려동물의 이전 업로드와 비교
- "3월 1일 대비 눈 충혈 60% 감소" 같은 개선도 표시
- 장기 차트 = 재방문 강력한 이유

### 도입 5: 수의사 아바타 브랜드
- 펫에토 대표 수의사 1명을 전면에 (가상 or 실제)
- `/for-vets` 첫 전문가를 "펫에토 수석 수의사 김○○" 형태로 캐릭터화
- 콘텐츠·답변·영상 전부 이 아바타로 통일
