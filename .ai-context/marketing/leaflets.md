> 크기 기본: **A5(148×210mm)** 또는 **DL 3단 접지(99×210mm ×3)**

## 🎯 3 타겟별 핵심 메시지 (모든 컨셉 공통)

### 타겟 A: 수의사 (개인)
- **Pain**: "SNS에 반려인들이 올리는 오진 위험 글들이 안타깝다. 내 전공을 살려 답하고 싶지만 플랫폼이 없다."
- **Hook**: "환자 케이스 답변하고 포인트·뱃지 받는 수의학 전문가 네트워크"
- **Call to Action**: "전문가 등록 → 케이스 답변 → 보상"
- **가치 제안 3가지**:
  1. 실제 학술 경력 증명 네트워크
  2. 답변 건수 = 포인트 = 현금화 가능
  3. 답변자 정보 100% 공개 (이름·병원·학교) = 개인 브랜딩

### 타겟 B: 동물병원 원장
- **Pain**: "신환 유입이 SNS/블로그 마케팅만으론 한계. 광고비 대비 전환율 낮음."
- **Hook**: "병원 브랜딩 + 전국 반려인 리드 = 신환 채널"
- **Call to Action**: "병원 계정 등록 → AI 선별된 환자 자동 매칭"
- **가치 제안 3가지**:
  1. AI가 심각도 분류 → 적합한 케이스만 병원에 라우팅
  2. 병원 프로필 페이지 + 수의사 답변 누적 = 신뢰 자산
  3. 상담·예약 전환 지표 대시보드

### 타겟 C: 펫샵
- **Pain**: "우리 가게 반려동물 홍보할 채널이 네이버 블로그/인스타뿐. 신뢰성 떨어짐."
- **Hook**: "반려동물 올리면 AI가 건강체크 해주고, 그 콘텐츠가 펫에토 피드에 노출"
- **Call to Action**: "펫샵 파트너 등록 → 피드 업로드 → AI 검증 뱃지 + 바이럴"
- **가치 제안 3가지**:
  1. 입양 전 AI 건강 검증 = 신뢰 마크
  2. 펫에토 피드 = 반려인 모이는 곳 = 노출 기회
  3. 파트너 펫샵 뱃지 = 차별화 (동네 다른 펫샵 대비)

## 📐 Concept 1 — "Editorial One-Pager"
*앤트로픽 에디토리얼 감성 · 잡지 한 페이지 느낌*

### 판형
**A5 양면 / 낱장** (접지 X, 한 장짜리)

### 타겟 A (수의사) 카피
- 앞면 헤드라인: **"수의사의 답은 어디에 있는가?"**
- 3 Reasons:
  1. **증빙된 경력** — 수의사 면허·학교·전공 인증. 답변마다 "누가" 명확하게.
  2. **답변이 자산** — 매 답변이 개인 브랜딩이자 포인트. 현금화 가능.
  3. **AI가 선별** — 스팸/비전문 질문은 AI가 거름. 수의학적으로 유의미한 케이스만.

### 타겟 B (원장) 카피
- 앞면 헤드라인: **"병원이 머무는 곳, 환자가 모이는 곳"**
- 3 Reasons:
  1. **AI 라우팅** — 지역·증상 매칭된 환자 자동 리드.
  2. **병원 자산화** — 수의사 답변 누적이 병원 신뢰도로 남음.
  3. **투명 지표** — 유입·상담 전환 대시보드 무료.

### 타겟 C (펫샵) 카피
- 앞면 헤드라인: **"믿고 데려가는 반려동물"**
- 3 Reasons:
  1. **AI 건강 검증** — 입양 전 AI 체크로 "우리 가게" 신뢰 마크.
  2. **피드 노출** — 펫에토 피드 = 반려인 집결지 = 무광고 바이럴.
  3. **파트너 뱃지** — 차별화된 정체성. 주변 펫샵과 구분.

### 사양
- 종이: **스노우화이트 200g** (고급 매트)
- 인쇄: 양면 4도
- 단가(500장): 10~15만원

## 📇 Concept 4 — "Story Card Series (이야기 카드 시리즈)"
*4장 미니 카드 세트 + 벨리밴드*

### 판형
**90×140mm 카드 4장 + 종이 벨리밴드로 묶음** — 선물 느낌

### 4장 구성
```
[CARD 1 - Hook]        [CARD 2 - How]        [CARD 3 - Why]         [CARD 4 - CTA]
┌─────────────┐        ┌─────────────┐       ┌─────────────┐        ┌─────────────┐
│             │        │             │       │             │        │             │
│  한 문장     │        │  사용법      │       │  증빙        │        │  QR          │
│  큰 후킹     │        │  3 Steps    │       │  "누구가"   │        │  지금        │
│             │        │             │       │  답하는가    │        │  시작        │
│             │        │  Step 1     │       │             │        │             │
│  pet-eto    │        │  Step 2     │       │  수의사 명단 │        │             │
│             │        │  Step 3     │       │  (일부 노출) │        │  pet-eto    │
└─────────────┘        └─────────────┘       └─────────────┘        └─────────────┘
   Terra Warm            Cream                Cream                   Ink Deep
```

### 타겟 A (수의사) 카드별 카피
- **CARD 1 Hook**: *"내 전공이, 반려인의 안심이 됩니다."*
- **CARD 2 How**: 등록 → 케이스 수신 → 답변 → 포인트
- **CARD 3 Why**: "답변 수의사 100% 공개 — 당신의 커리어가 됩니다"
- **CARD 4 CTA**: QR → 지금 5분 등록

### 타겟 B (원장) 카드별 카피
- **CARD 1 Hook**: *"광고 대신, 수의학이 환자를 데려옵니다."*
- **CARD 2 How**: 병원 등록 → AI가 케이스 매칭 → 상담 예약 전환
- **CARD 3 Why**: "전국 반려인이 답변자로 당신의 병원을 기억합니다"
- **CARD 4 CTA**: QR → 병원 파트너 신청

### 타겟 C (펫샵) 카드별 카피
- **CARD 1 Hook**: *"입양 전, AI가 먼저 건강체크 합니다."*
- **CARD 2 How**: 파트너 등록 → 반려동물 업로드 → AI 검증 뱃지
- **CARD 3 Why**: "파트너 펫샵 = 펫에토 반려인이 신뢰하는 가게"
- **CARD 4 CTA**: QR → 파트너 등록

### 사양
- 종이: 카드 **아르떼 260g** × 4장 / 벨리밴드 **크라프트 150g**
- 인쇄: 양면 4도
- 후가공: 벨리밴드에 펫에토 심볼 레터프레스
- 단가(500 세트): 35~50만원

### ⭐ 이 컨셉의 강점
**체험이 된다**. 수의사가 벨리밴드 풀면서 카드 한 장씩 보는 경험 자체가 펫에토 철학을 전달.
