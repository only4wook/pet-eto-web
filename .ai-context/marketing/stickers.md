## 🏅 TYPE 1 — "펫에토 파트너 뱃지"
*병원·펫샵 카운터/출입문에 붙이는 인증 뱃지*

### 용도
파트너십 체결한 병원/펫샵이 고객에게 "우리는 펫에토 파트너"임을 시각화.  
→ 차별화 도구 + 바이럴 증폭 + 페이드IN 마케팅

### 디자인 변형 3가지

**A. 수의사·병원 파트너 (원형)**
```
        ┌───────────┐
       / pet-eto    \
      /   certified  \
     |   partner      |
      \   hospital   /
       \   ──────── /
        └───────────┘
```
- 지름 80mm
- 원형 테두리 이중선 (Ink Deep 0.5pt + Terra Warm 1pt)
- 중앙 텍스트: Serif (Anthropic 스타일)
- 하단 "hospital" 라벨
- 색: 크림 배경 + Terra Warm 강조

**B. 펫샵 파트너 (직사각형)**
```
  ┌─────────────────────────┐
  │  pet-eto · 파트너 펫샵   │
  │  ─────                  │
  │  AI 건강 검증 완료        │
  └─────────────────────────┘
```
- 100×50mm
- 가로형, 쇼윈도우에 부착 용이
- "AI 건강 검증 완료" 강조

**C. 수의학 전문가 개인 뱃지 (육각형)**
```
         ┌─────┐
        /       \
       /  pet-   \
      |   eto    |
       \ expert /
        \______/
```
- 지름 60mm
- 육각형 (벌집/의학 메타포)
- 수의사 개인 병원 벽에 부착 (프로필 인증용)

### 사양
- 재료: **리무벌 PVC + UV 코팅** (야외 내구성 2년)
- 접착: 재부착 가능 (리무벌) — 이전 시 자국 無
- 인쇄: 4도 디지털 + UV 코팅
- 포장: 반투명 파우치 (택배 발송 대비)

### 단가 (500장 기준)
- Type A: 약 15-20만원
- Type B: 약 12-18만원
- Type C: 약 10-15만원

## 📱 TYPE 2 — "QR 스티커"
*영업킷·리플릿에 동봉. 떼어서 본인 노트북·다이어리에 붙이게*

### 디자인
```
┌──────────────────────┐
│                      │
│   pet-eto            │
│   ────               │
│                      │
│   ┌────────────┐     │
│   │            │     │
│   │    QR      │     │
│   │            │     │
│   └────────────┘     │
│                      │
│   스캔하면 바로       │
│                      │
└──────────────────────┘
```
- 크기: 50×70mm (노트북에 적당)
- 배경: Cream
- 테두리: 은은한 Terra Warm 헤어라인

### 변형
- **v1. 수의사용**: "답변하러 가기" 카피
- **v2. 원장용**: "병원 등록하기"
- **v3. 펫샵용**: "펫샵 파트너 신청"
- **v4. 일반용**: "펫에토 바로 체험"

### 사양
- 재료: **유포지 + 매트 라미네이팅**
- 접착: 일반 다이컷 접착
- 인쇄: 2도 (Ink Deep + Terra Warm) + QR 검정
- 단가(1000장): 10-15만원

## 🔖 TYPE 3 — "로고 스티커" (심플)
*증정용·콜렉션용. 펫오너가 지갑이나 다이어리에 붙이고 싶어지게*

### 디자인 변형 3가지

**A. 미니 로고 (원형)**
- 지름 30mm
- 크림 배경 + Terra 로고
- 오브젝트로서 완결됨

**B. 워드마크 (가로형)**
- 60×20mm
- "pet-eto" 워드마크만
- 블랙 배경 + 크림 로고 (vs 크림 + 잉크 2버전)

**C. 발자국 심볼 (투명)**
- 지름 25mm
- 투명 배경 + Terra 발자국 실루엣
- 노트북·텀블러·어디든 녹아듦

### 사양
- 재료: **유포지** (매트) + **투명 PVC** (C만)
- 인쇄: 2-4도
- 단가(1000장 5종 믹스): 15-20만원

### 용도

## 🎯 TYPE 5 — "수의사 이름 개인 뱃지 (수의사 맞춤)"
*1:1 영업용. 수의사 이름을 인쇄해서 선물*

### 컨셉
```
  ┌─────────────────────┐
  │                     │
  │  김수의 수의사       │
  │  (답변 수의사 뱃지)   │
  │                     │
  │  pet-eto verified   │
  │                     │
  └─────────────────────┘
```
- 크기: 100×60mm
- 재료: 유포지 + 매트 라미네이팅
- 인쇄: 2도 (타겟 수의사 이름 가변 인쇄)

### 운영 방식
- 1차 영업 미팅 후, 해당 수의사 이름 넣어 인쇄 → 2차 방문 시 선물
- "직접 만들어 드렸어요" — 심리적 유대 형성

### 사양
- 소량 인쇄 (30-50장씩)
- 단가(50장): 약 5만원 (디지털 가변 인쇄)

## 💡 스티커 활용 전략

### 1. **파트너 뱃지는 무료 제공** (영업 결재용)
- 파트너 계약 체결 시 Type 1 뱃지 10장 무상 제공
- 병원 입구·카운터·대기실에 부착 권장
- 부착 사진 찍어서 펫에토 SNS에 공유 (상호 바이럴)

### 2. **QR 스티커는 명함 보완재**
- 명함 받기 애매한 상대 (예: 수의대 학생)에게
- QR만으로도 체험 유도

### 3. **로고 스티커는 브랜드 자산 확장**
- 팬덤화 (수의사가 자기 다이어리에 붙임)
- 굿즈 상품화 가능성

### 4. **캠페인 스티커는 이벤트 무기**
- 수의학 컨퍼런스·학회 방문 시 1000장씩 배포
- "이거 재밌어요" → 대화 트리거

### 5. **개인 뱃지는 최강의 1:1 영업 도구**
- 수의사 입장: "내 이름이 새겨진 게 왔다" → 각인 효과
- 비용 대비 전환율 최고
