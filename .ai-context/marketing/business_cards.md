## 🏛 Concept 1 — "Apple Minimalist"
*by 애플 수석 디자이너 관점 · Jony Ive 유산*

### 철학
극도의 미니멀. 명함 한 장이 "하나의 오브젝트". 만졌을 때 촉감이 디자인의 일부. 정보는 타이포그래피로 말한다.

### 앞면 레이아웃
```
┌─────────────────────────────────────┐
│                                     │
│  pet-eto                            │
│                                     │
│                                     │
│                    권은환           │
│                    대표             │
│                                     │
│                                     │
│  only4wook@gmail.com   010-XXXX-XXXX│
└─────────────────────────────────────┘
```
- 배경: **Cream Paper (#F5F0E8)**
- 좌상단 로고: **Ink Deep**, 12mm 폭, 소문자 소서체
- 우측 이름: 센터 정렬, Pretendard Bold 16pt
- 하단 연락처: Pretendard Regular 7pt, 자간 넓게 (tracking +40)

### 뒷면 레이아웃
```
┌─────────────────────────────────────┐
│                                     │
│                                     │
│             ▓▓▓▓▓▓▓▓▓▓              │
│             ▓▓QR CODE▓▓             │
│             ▓▓ 2cm×2cm▓▓            │
│             ▓▓▓▓▓▓▓▓▓▓              │
│                                     │
│          pet-eto.vercel.app         │
└─────────────────────────────────────┘
```
- 완전히 비어있는 크림 바탕
- 정중앙 QR (Ink Deep on Cream)
- QR 아래 URL 6pt

### 사양
- 종이: **몽블랑 260g** (고급 매트)
- 후가공: **로고만 엠보싱** (Blind Deboss, 색 없이 눌림)
- 인쇄: 양면 1도 (잉크 딥 단색)
- 단가 추정: 200장 8-12만원

### 수의사에게 주는 느낌
"이 사람 진지하다. 디자인에 투자하는 스타트업이라는 건 디테일을 안다는 뜻."

## 📖 Concept 3 — "Anthropic Editorial"
*by 앤트로픽 수석 디자이너 관점 · Jenny Wen 감성*

### 철학
연구소의 명함. 지적이지만 따뜻하다. 세리프 활자가 "우리는 생각하는 회사"라고 말한다. Anthropic의 크래프트 + Letterform에 대한 경의.

### 앞면 레이아웃
```
┌─────────────────────────────────────┐
│                                     │
│  pet-eto                      ─     │
│  ─────────                          │
│                                     │
│   권 은 환                          │
│   Eunhwan Kwon                      │
│                                     │
│   Founder & CEO                     │
│                                     │
│ ─────────────────────────────────   │
│  only4wook@gmail.com  010-XXXX-XXXX │
└─────────────────────────────────────┘
```
- 배경: **Cream Paper** (촉감 있는 재질)
- 이름: **ITC Galliard Italic** 또는 **Source Serif Pro Semibold Italic** 18pt
- 이름 자간 넓게 (letter spacing 200)
- 한영 이름 병기, 영문은 작게
- 가는 가로선 2개가 구조 잡아줌 (헤어라인 0.5pt Terra Warm)
- 직함: Pretendard Regular 8pt
- 우상단 가로 짧은 선: Terra Warm

### 뒷면 레이아웃
```
┌─────────────────────────────────────┐
│                                     │
│  "흩어진 펫 정보를 수의사의          │
│   언어로 통합합니다."                │
│                                     │
│   — pet-eto manifesto              │
│                                     │
│                                     │
│                    ┌─────────┐      │
│                    │   QR    │      │
│                    └─────────┘      │
│  ─────────────────────────────────  │
│  Scan to experience · pet-eto.app   │
└─────────────────────────────────────┘
```
- 인용문: 세리프 Italic 10pt, 행간 넉넉
- QR 우하단 배치 (작게 1.5cm)
- 매니페스토 인용구 스타일이 "이 회사는 철학이 있다"를 전함

### 사양
- 종이: **빈티지 크림 250g** 또는 **반누보 내추럴 240g**
- 후가공: **레터프레스** (깊은 압인, 촉감)
- 인쇄: 양면 2도 (Ink Deep + Terra Warm)
- 단가 추정: 200장 15-25만원 (레터프레스 프리미엄)

### 수의사에게 주는 느낌
"이 회사, 책 같다. 깊이 있어 보여. 믿음이 간다."

## 📋 Concept 4 — "Vet Chart (한국적 의료 에디토리얼)"
*진료차트 메타포를 명함에 이식*

### 철학
수의사가 매일 보는 차트 스타일. "우리는 당신의 언어를 안다"는 공감의 디자인. 그리드가 곧 신뢰.

### 앞면 레이아웃
```
┌─────────────────────────────────────┐
│ PET-eto  │  진료 차트 No. 2026-001  │
│──────────┼──────────────────────────│
│ NAME     │ 권은환 Eunhwan Kwon      │
│──────────┼──────────────────────────│
│ ROLE     │ 대표 Founder             │
│──────────┼──────────────────────────│
│ CONTACT  │ only4wook@gmail.com      │
│          │ 010-XXXX-XXXX            │
│──────────┼──────────────────────────│
│ ISSUED   │ 2026.04.23  │ pet-eto    │
└─────────────────────────────────────┘
```
- 차트처럼 필드 라벨 + 값 구조
- 라벨: Monospace (JetBrains Mono 6pt)
- 값: Pretendard Regular 8-10pt
- 그리드 선: Ink Deep 0.3pt, 세미 투명 30%
- 상단 "진료 차트 No." → 재치있는 메타 장치 (일련번호 = 명함 배포 번호)
- 배경: **Cream Paper** (차트 종이 느낌)

### 뒷면 레이아웃
```
┌─────────────────────────────────────┐
│                                     │
│  처방 Rx                             │
│                                     │
│  반려동물 건강 AI 체크               │
│  1분 스캔 처방합니다                 │
│                                     │
│          ┌──────────┐               │
│          │          │               │
│          │   QR     │               │
│          │          │               │
│          └──────────┘               │
│                                     │
│  Signature _______________          │
│                                     │
└─────────────────────────────────────┘
```
- "처방 Rx" 필기체 느낌 (Caveat 또는 Nanum Pen)
- QR 중앙
- 하단 "Signature" 공란 → 수의사가 손으로 서명해서 환자에게 전달할 수도 있게

### 사양
- 종이: **아르떼 210g** (매트, 차트 느낌)
- 후가공: 없음 (차트스러움 유지)
- 인쇄: 양면 2도 (Ink Deep + Terra Warm 포인트)
- 단가 추정: 200장 6-10만원

### 수의사에게 주는 느낌
"이거 재밌다. 우리를 이해하고 있네. 놀이 감각이 있다."

### ⭐ 이 컨셉의 강점
수의사 타겟에게 가장 효과적. 명함 자체가 대화 소재가 됨.

## 🏆 Concept 5 — "Premium Tactile (촉감 프리미엄)"
*Apple의 재질 감각 + Anthropic의 편집 감성 + 한국 공예*

### 철학
명함이 장인 작품. 정보 최소화, 촉감 최대화. 받는 순간 "와" 소리 나오는 물성.

### 앞면 레이아웃
```
┌─────────────────────────────────────┐
│                                     │
│                                     │
│                                     │
│              pet-eto                │
│              ────────               │
│                                     │
│                                     │
│                                     │
│                                     │
└─────────────────────────────────────┘
```
- 완전한 흰 여백 + 로고 1개만
- 로고 **Terra Warm 박(foil stamping)** 처리
- 로고 아래 가는 선 하나 **Blind Deboss (무색 압인)**
- 극단적 미니멀 — 정보는 뒷면으로

### 뒷면 레이아웃
```
┌─────────────────────────────────────┐
│                                     │
│  권은환 │ Eunhwan Kwon              │
│  대표 · Founder                     │
│                                     │
│                                     │
│  only4wook@gmail.com                │
│  010-XXXX-XXXX                      │
│  pet-eto.vercel.app                 │
│                                     │
│                        ┌───────┐    │
│                        │  QR   │    │
│                        └───────┘    │
└─────────────────────────────────────┘
```
- 좌측 상단: 이름 (세리프, 크게)
- 본문 정보: 작게, 왼쪽 정렬
- 우하단 QR (작게 1.5cm)
- 전체 크림 배경

### 사양
- 종이: **반누보 내추럴 300g** 또는 **크라운 260g**
- 후가공:
  - 앞면: Terra Warm **박(Foil Stamp)** + Blind Deboss 라인
  - 뒷면: Emboss 엣지 (테두리 1mm 무색 압인)
- 인쇄: 뒷면만 1도 인쇄 (Ink Deep) + 양면 후가공
- 단가 추정: 200장 25-40만원 (후가공 프리미엄)

### 수의사에게 주는 느낌
"이거 장난 아니네. 이 명함 받는 것만으로도 기분 좋다. 이 사람 만나야겠다."

## 🔁 공통 제작 체크리스트

인쇄소 발주 시 반드시 확인:
- [ ] 출혈(bleed) 3mm 포함했는가
- [ ] 안전여백(safe area) 5mm 확보했는가
- [ ] 폰트 아웃라인 처리(깨짐 방지)
- [ ] CMYK 컬러 모드 (RGB 금지)
- [ ] 해상도 300dpi 이상
- [ ] QR 실제 스캔 테스트 (인쇄 샘플에서)
- [ ] 종이 샘플 먼저 받아보기 (무광/유광 차이)
- [ ] Pantone 넘버 지정
- [ ] 교정지 2회 (컬러 교정 → 최종 교정)
