# P.E.T 파트너 면접 가이드

---

## 면접 방식

| 방식 | 대상 | 도구 |
|------|------|------|
| **대면 면접** | 고양시/일산/서울 서북권 거주자 | 카페에서 30분 |
| **화상 면접** | 원거리 / 일정 조율 어려운 경우 | 카카오톡 영상통화 or 줌 |

---

## 면접 체크리스트 (30분)

### 1단계: 기본 확인 (5분)
- [ ] 신분증 확인 (대면: 실물 / 화상: 화면 보여주기)
- [ ] 신청서 내용 사실 확인
- [ ] 활동 가능 시간대/요일 확인

### 2단계: 역량 확인 (10분)
| 질문 | 확인 포인트 |
|------|-----------|
| "반려동물 양육 경험을 말씀해주세요" | 실제 경험 깊이 |
| "급식/산책/투약 경험이 있나요?" | 실무 능력 |
| "응급 상황 시 어떻게 대처하시겠어요?" | 위기 대응력 |
| "강아지/고양이가 물려고 하면?" | 동물 행동 이해도 |
| "보호자가 불만을 제기하면?" | 서비스 마인드 |

### 3단계: 상황 시뮬레이션 (10분)
```
상황 1: "케어 중 강아지가 갑자기 구토를 했습니다. 어떻게 하시겠어요?"

모범 답변 포인트:
✅ 토한 내용물 사진 촬영
✅ 보호자에게 즉시 연락
✅ 추가 증상(무기력, 설사) 관찰
✅ 심하면 가까운 병원 이송 제안

상황 2: "고객 집 열쇠를 받았는데 고양이가 숨어서 안 나옵니다."

모범 답변 포인트:
✅ 억지로 꺼내지 않음
✅ 간식/사료를 근처에 놓고 기다림
✅ 보호자에게 상황 보고
✅ 화장실/급수 확인
```

### 4단계: 안내 및 계약 (5분)
- [ ] 수수료 구조 설명 (파트너 75%, 원천징수 3.3%)
- [ ] 서비스 매뉴얼 전달
- [ ] 보험 가입 안내
- [ ] 시범 케어 일정 잡기

---

## 합격/불합격 기준

### 합격 ✅
- 반려동물 양육 경험 1년+
- 응급 상황 대처 이해
- 서비스 마인드 (친절, 책임감)
- 시간 약속 준수 의지

### 불합격 ❌
- 반려동물 경험 전무
- 응급 상황에 대한 인식 부족
- 무책임한 태도
- 신원 확인 거부

---

## 화상 면접 세팅

### 카카오톡 영상통화:
1. 파트너에게 카카오톡 친구 추가 요청
2. 약속 시간에 영상통화 발신
3. 신분증은 카메라에 비춰달라고 요청
4. 녹화는 하지 않음 (개인정보)

### 줌(Zoom):
1. 무료 계정으로 회의 링크 생성
2. 파트너에게 링크 전송
3. 40분 제한 (무료), 면접은 30분이면 충분

---

## 시범 케어 (합격 후)

- 대표님이 동행하여 1회 시범 케어 진행
- 실제 고객 또는 대표님의 반려동물로 테스트
- 체크 항목:
  - [ ] 시간 약속 준수
  - [ ] 동물에 대한 태도
  - [ ] 사진/영상 보고 능력
  - [ ] 급식/급수 정확성
  - [ ] 집 정리 상태
