## 1. P.E.T는 무엇인가?

**한 줄 정의:** 반려동물 보호자의 '시간 공백'을 해결하는 긴급 컨시어지 O2O 플랫폼

**핵심 가치:** "갑자기 못 돌볼 때, 10분 안에 믿을 수 있는 케어러를 연결해드려요"

**경쟁사와의 차이점:**
- 핏펫 = 건강/의료 데이터 → 커머스 (물건을 판다)
- 펫프렌즈 = 빠른 배송 → 커머스 (물건을 배달한다)
- **P.E.T = 시간/돌봄 공백 → 사람을 연결한다 (서비스를 중개한다)**

## 2. TAM / SAM / SOM

### TAM (Total Addressable Market) — 전체 시장
- 국내 반려동물 시장 규모: **약 10조 원** (2025년 기준, 농림축산식품부)
- 반려가구: **약 602만 가구** (4가구 중 1가구)
- 반려동물 관련 서비스(돌봄/미용/호텔/훈련) 시장: **약 2조 원**

### SAM (Serviceable Available Market) — 접근 가능 시장
- 1~2인 가구 반려인 (긴급 돌봄 수요 핵심 타겟): **약 250만 가구**
- 이 중 서울/수도권: **약 150만 가구**
- 연간 1회 이상 긴급 돌봄이 필요한 가구: **약 60만 가구** (40% 추정)
- 건당 평균 5만원 기준: **약 3,000억 원 시장**

### SOM (Serviceable Obtainable Market) — 초기 목표 시장
- **고양시 + 파주 + 서울 서북권** (대표님 거주/활동 권역)
- 해당 지역 1~2인 반려가구: **약 15만 가구**
- 초기 1년 목표 고객: **500명** (월 40~50명 신규)
- 연간 매출 목표: 500명 × 연 3회 이용 × 5만원 = **약 7,500만원**

## 3. 현재 잘하고 있는 것 ✅

### 3-1. 제품(MVP)
| 항목 | 상태 | 평가 |
|------|------|------|
| 웹 플랫폼 (pet-eto.vercel.app) | ✅ 라이브 | 무료 배포, 24시간 운영 |
| AI 증상 분석 (텍스트 기반) | ✅ 21개 카테고리, 300+ 키워드 | 경쟁사에 없는 독자 기능 |
| 펫-위키 (28종 상세 정보) | ✅ 질병+비용+관리법+초보가이드 | 콘텐츠 경쟁력 높음 |
| 커뮤니티 (게시판+피드) | ✅ 카테고리 7종 | 후기/문의 게시판 포함 |
| 카카오톡 채널 연동 | ✅ 플로팅 버튼 | 1:1 상담 가능 |
| 관리자 대시보드 | ✅ 유저/글/분석/위키 관리 | 운영 도구 갖춤 |

### 3-2. 전략
- **핵심 페인포인트 정확히 짚음**: 1~2인 가구의 긴급 돌봄 공백
- **노코드/MVP 접근**: 큰 투자 없이 빠르게 실행
- **카톡 기반 수동 매칭**: 자동화 전에 시장 검증 (린스타트업 방식)
- **한양대 창업동아리**: 학교 자원 활용 가능

## 4. 현재 부족한 것 ⚠️

### 4-1. 실적/데이터 부재 (가장 큰 약점)
- 실제 매칭 건수: **0건** (아직 서비스 시작 전)
- 유료 고객: **0명**
- 후기/사례: **0개**
- → **투자자/심사역이 가장 먼저 묻는 것**: "실제로 써본 사람이 있나요?"

### 4-2. 수익 모델 불명확
- 현재 수익: **0원** (무료 서비스)
- 수수료 구조 미정: 건당? 월정액? 비율?
- → 최소 1건이라도 유료 거래 실적이 필요

### 4-3. 파트너(펫시터) 네트워크 미구축
- 검증된 파트너: **0명** (3단계 검증 프로세스는 설계됨)
- → 서비스를 시작하려면 최소 3~5명의 파트너 필요

### 4-4. 팀 역량 증명
- 개발: Claude AI 의존 (자체 개발 역량 부족)
- 수의학 전문성: 자문단 미구성
- → 심사 시 "이 팀이 왜 이 사업을 할 수 있는가?" 답변 필요

### 4-5. 법적 준비
- 동물 관련 사업자 등록: 미완
- 보험: 케어 중 사고 시 배상 책임 보험 미가입
- → 서비스 전 필수 준비

## 6. P.E.T만의 독자적 경쟁력 (핏펫·펫프렌즈와 다른 것)

### 6-1. 무재고 모델 (Asset-light)
- 핏펫: 사료 재고 필요 → 수십억 투자
- 펫프렌즈: 물류 창고 필요 → 수백억 투자
- **P.E.T: 사람만 연결 → 초기 투자 거의 0원**

### 6-2. AI 기반 사전 상담
- 웹사이트에서 증상 입력 → AI가 즉시 분석 → 병원 안내
- **경쟁사에 없는 기능**: 핏펫은 검사 키트 필요, P.E.T는 텍스트만으로 무료 분석
- 이 AI 상담 데이터가 쌓이면 → 맞춤형 서비스 추천 가능

### 6-3. 지역 밀착형 신뢰
- 대형 앱은 전국 단위 → 신뢰 구축 어려움
- **P.E.T는 고양시에서 시작 → 동네 사람끼리 신뢰** → 입소문 마케팅
- 대표가 직접 파트너를 면접 → "이 사람이 직접 뽑은 시터"라는 신뢰

### 6-4. 커뮤니티 + 서비스 결합
- 위키로 유입 → 커뮤니티로 정착 → 긴급 돌봄 시 자연스럽게 서비스 이용
- **콘텐츠 → 트래픽 → 전환**이라는 성장 플라이휠
